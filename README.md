# 🧮 Tkinter Calculator App

A simple calculator built with **Python Tkinter** that supports basic arithmetic operations.  
This project demonstrates GUI development using Tkinter’s `Entry`, `Button`, and `Grid` layout system.

---

## 🚀 Features
- Interactive number pad (0–9)
- Supports operations: `+`, `-`, `*`, `/`
- Decimal point support
- Clear (`C`) button to reset input
- Equals (`=`) button to calculate result
- Error handling for invalid inputs
- Responsive grid layout with column/row expansion
